# Changelog

本项目的重要变更都会记录在此文件中。

本文件格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

> 说明：当前目录不是 Git 仓库，以下内容根据本轮实际开发结果补录，暂未提供版本比较链接。

## [Unreleased]

## [0.1.3] - 2026-04-08

### Added

- 新增页面版本信息栏，显示当前版本、更新状态、端口号、会话目录与 Release 页面链接。
- 新增后端 `/api/check_update` 接口，用于检查 GitHub 最新 Release，并返回版本、服务端口与会话目录信息。
- 新增会话收藏、标签与“只看收藏”筛选，数据持久化保存在本地。
- 新增筛选列表与单会话导出能力，支持导出为 JSON / Markdown。
- 新增页头 GitHub 图标入口与右下角“回到顶部”悬浮按钮。
- 新增快捷键帮助浮层，支持 `?` 查看常用操作说明。
- 新增轻量 Toast 提示，替代原生 `alert()` 反馈。
- 新增批量选择模式，支持全选、批量导出、批量收藏/取消收藏与批量删除。
- 新增主题自动模式，可跟随系统深浅色设置。
- 新增会话卡片拖拽整理与分组视图，支持本地持久化自定义布局。
- 新增 Analytics 消息趋势图，支持按日 / 按周查看趋势。

### Changed

- 前端启动时自动检查是否存在新版本；如有更新，会在页面中显示可点击的更新提示；当前已是最新版本时不再重复显示“最新发布版本号”。
- GitHub 仓库链接从页脚移动到左上角图标按钮，减少页脚信息重复。
- 会话搜索已支持高亮标题、路径、预览与标签中的匹配内容。
- 会话列表在大量数据场景下启用虚拟滚动，减少一次性渲染压力。

## [0.2.0] - 2026-04-06

### Added

- 新增详情弹窗“用户指令”摘要区，支持快速预览并跳转到原始消息位置。

### Changed

- 优化用户指令摘要区交互：默认折叠、折叠时显示首条指令、多条指令时才显示展开按钮。
- 优化初始加载性能：会话扫描增加摘要缓存，分析页改为按需渲染，减少首屏阻塞。
- 优化详情弹窗与确认弹窗样式，统一遮罩、面板、按钮反馈与移动端布局。
- 优化会话搜索、详情读取与 CLI 命令生成流程，减少重复全量扫描。
- 更新三平台运行版 ZIP 交付物，重新打包最新 macOS / Windows / Linux 运行包。

### Fixed

- 修复带 `<system-reminder>` / `<system-notification>` 的用户输入在详情弹窗中被错误过滤的问题。
- 修复部分序列化文本载荷（如 `{"type":"text","text":"..."}`）未被正确还原为用户指令的问题。
- 修复会话文件读取、删除与 CLI 启动路径缺少约束的问题，阻止非法路径访问。

## [0.1.0] - 2026-04-05

### Added

- 新增根目录 `AGENTS.md` 中文协作指南，明确项目结构、开发命令、测试方式与安全约束。
- 新增会话卡片双行预览，支持展示首条用户消息与最近一条有效消息摘要。
- 新增路径别名功能，可在前端为工作区路径设置自定义显示名称，并支持重置。
- 新增详情弹窗消息搜索、高亮、仅看命中项、长消息折叠/展开能力。
- 新增工具调用详情展示，可查看工具名称、操作摘要、输入预览与工具结果状态。
- 新增 3 个运行版交付包：
  - `droid-packages/FactoryDroidSession-Manager-runtime-macos.zip`
  - `droid-packages/FactoryDroidSession-Manager-runtime-windows.zip`
  - `droid-packages/FactoryDroidSession-Manager-runtime-linux.zip`

### Changed

- 优化工作区路径显示逻辑，优先使用真实 `cwd`，避免错误地把连字符路径转换成斜杠路径。
- 优化特殊工作区名的路径解码逻辑，在 `cwd` 为 `.` 时可回推出更合理的实际路径。
- 优化会话详情中连续工具消息的展示方式，支持按组汇总并按需展开查看。
- 优化详情弹窗视觉样式，包括头部、信息卡片、消息卡片、工具卡片、搜索区与移动端布局。
- 优化弹窗在深色 / 浅色主题下的视觉一致性，统一使用主题变量控制边框、阴影、渐变与代码块背景。
- 补充当前变更记录为更标准的 Changelog 结构，便于后续持续维护。

### Fixed

- 修复部分同名会话工作区路径在前端显示错误的问题。
- 修复详情弹窗搜索框输入时因整块重渲染导致的焦点丢失问题。
- 修复工具结果消息缺少独立展示的问题，使执行结果更容易定位。
